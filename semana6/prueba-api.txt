Al enviar una petición POST a la ruta http://localhost:3000/registro con el siguiente JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

el servidor recibe los datos enviados por el cliente a través de req.body y responde con un objeto JSON que confirma que la información fue recibida correctamente.

Respuesta obtenida:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto demuestra que la ruta POST funciona correctamente y que Express puede procesar datos enviados en formato JSON.
